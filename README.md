# Guia Git Colaborativo
Este repositório contém um resumo colaborativo dos principais conceitos de Git e GitHub.
## Conceitos Básicos de Git
1.- Repositório
É um local que fica salvo todo o projeto e as atualizações no desenvolvimento do mesmo
2.- Commit
É um código de programação para registrar alterações em arquivos 
3.- Branch
É como um rascunho idependente  onde se houver erro não vai alterar o projeto principal
4.- Merge
É um código para enviar um Branch para o main
